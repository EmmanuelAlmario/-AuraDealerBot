# -AuraDealerBot
Bot de discord.
Tecnologias a usar:
Python:
  Librerias:
      -discord.py

Funcionalidades:
1. Da o elimina "Aura" dependiendo de las estadisticas que tenga un jugador en la partida.
2. Crea una tabla en un canal especifico en el cual muestra la tabla de quien tenga mas aura hasta el de menos.
3. Muestra el peor de la partida. 
4.  Niveles dependiendo de su cantidad de aura.
5. Al final de cada mes crea una imagen de quien fue el que tuvo mas aura.
6. Agregar misiones para farmear "Aura".
